# Texture

A **Texture** is a structured collection of data, typically a 2D image (texels). In D3D11, textures are general-purpose multidimensional arrays — they can store colors, normals, depth values, or any float data.

关联：[[D3D11.BackBuffer]] [[D3D11.DepthBuffer]]

---

## Texture 只是数据容器

`Texture.Sample()` 永远返回 `float4`（四个浮点数），**GPU 不知道这代表颜色还是法线**。语义由程序员约定：

```hlsl
// 采样同一份数据，意义完全取决于你怎么解读
float3 albedo    = AlbedoTex.Sample(s, uv).rgb;       // 当颜色用
float3 normal    = NormalTex.Sample(s, uv).rgb * 2 - 1; // 当法线用
float  depth     = DepthTex.Sample(s, uv).r;           // 当深度用
```

## SamplerState（采样规则）

Texture 存数据，**SamplerState 决定怎么读这个数据**——当 UV 落在两个像素之间，或者超出 0~1 范围时，该怎么处理。

```hlsl
SamplerState ColorSampler
{
    Filter   = MIN_MAG_MIP_LINEAR;
    AddressU = WRAP;
    AddressV = WRAP;
};
```

### Filter（过滤方式）

UV 坐标是连续的小数，Texel 是离散的格子，UV 落在两个 Texel 之间时：

| Filter | 效果 |
| :--- | :--- |
| `MIN_MAG_MIP_LINEAR` | 线性插值，平滑，无锯齿 |
| `MIN_MAG_MIP_POINT` | 最近邻，像素化/马赛克 |
| `ANISOTROPIC` | 各向异性，斜视角最清晰，开销最大 |

```
LINEAR（平滑）：    POINT（像素化）：
  ░░▒▒▓▓██           ░░░░▓▓▓▓
  ░░▒▒▓▓██           ░░░░▓▓▓▓
  平滑过渡            硬边缘
```

三个字母 `MIN_MAG_MIP` 分别对应：
- **MIN**：纹理缩小时（远处）
- **MAG**：纹理放大时（近处）
- **MIP**：Mipmap 层级切换时

### AddressMode（寻址模式）

UV 超出 0~1 范围时：

| 模式 | 效果 | 适用场景 |
| :--- | :--- | :--- |
| `WRAP` | 循环平铺 | 地板、砖墙等重复纹理 |
| `CLAMP` | 拉伸边缘像素 | UI、精灵图 |
| `MIRROR` | 每次越界镜像翻转 | 对称图案 |
| `BORDER` | 填充指定边框颜色 | Shadow Map 采样越界 |

## Texture 与 Sampler 分离

同一张纹理可以用不同规则采样，同一个 Sampler 可以用于不同纹理：

```hlsl
// 同一张纹理，两种效果
float4 smooth  = MyTex.Sample(LinearSampler, uv);
float4 pixelated = MyTex.Sample(PointSampler, uv);

// 不同纹理，复用同一个 Sampler
float4 albedo  = AlbedoTex.Sample(ColorSampler, uv);
float4 normal  = NormalTex.Sample(ColorSampler, uv);
```

## D3D11 绑定方式

```cpp
// Texture 绑定到 t0 槽
context->PSSetShaderResources(0, 1, &textureSRV);

// Sampler 绑定到 s0 槽
context->PSSetSamplers(0, 1, &samplerState);
```

```hlsl
// HLSL 对应声明
Texture2D  MyTexture : register(t0);
SamplerState MySampler : register(s0);

float4 color = MyTexture.Sample(MySampler, uv);
```

---

## Texture 的双重用途

同一张纹理可以作为输入（SRV）也可以作为输出（RTV），参见 [[D3D11.RenderTarget]]：

```
作为 SRV → 在 Shader 里采样（读）
作为 RTV → 作为 Render Target 被写入（写）

⚠️ 不能同时读写，切换前必须解绑
```

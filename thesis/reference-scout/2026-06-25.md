# Reference Scout — 2026-06-25

## Search Focus

当前 thesis 方向：**基于动画驱动的第三人称近战系统，研究如何通过 root motion / animation events / input & hit detection / parry / boss AI，让 Souls-like boss fight 既有动画重量感又保持响应性。**

---

## Top References

### 1. *Designing Game Feel: A Survey* (2020)

- **Type:** survey paper
- **Authors / Source:** Martin Pichlmair, Mads Johansen
- **Link:** https://arxiv.org/abs/2011.09201
- **Thesis fit:** Core
- **Supports:** Game Feel / Combat / Animation
- **Why it matters:**
  - 给 “game feel” 提供学术化框架。
  - 将 game feel 分成 physicality、amplification、support。
  - 可用于解释 animation commitment 与 input responsiveness 的张力。
- **How to use in thesis:** Literature Review 中定义 “responsive but weighty combat feel”。
- **Caveat:** 不是专门讲 melee combat，但适合作为理论框架。

### 2. *What Features Influence Impact Feel? A Study of Impact Feedback in Action Games* (2022)

- **Type:** paper
- **Authors / Source:** Zhonghao Lin, Haihan Duan, Zikai Alex Wen, Wei Cai
- **Link:** https://arxiv.org/abs/2208.06155
- **Thesis fit:** Core
- **Supports:** Combat / Game Feel / Hit Feedback
- **Why it matters:**
  - 直接研究 action games 的打击感。
  - 提出 impact feedback 的 feature framework。
  - 可支撑 hit stop、sound、camera shake、hit reaction 对近战体验的影响。
- **How to use in thesis:** Combat feedback / evaluation section。
- **Caveat:** 偏 feedback 分析，不直接教 hit detection 实现。

### 3. *Game Feel: A Game Designer’s Guide to Virtual Sensation* (2008)

- **Type:** book
- **Author / Source:** Steve Swink, CRC Press
- **Link:** https://www.game-feel.com/
- **Thesis fit:** Core
- **Supports:** Game Feel / Input Responsiveness
- **Why it matters:**
  - 经典 game feel 书。
  - 对 “input → response → perceived control” 很有帮助。
  - 适合解释为什么玩家觉得 combat “responsive”。
- **How to use in thesis:** Design theory 背景。
- **Caveat:** 不是 peer-reviewed paper，但在 game design 领域引用价值高。

### 4. *Motion Matching for Character Animation and Virtual Reality Avatars in Unity* (2023)

- **Type:** thesis / technical research
- **Author / Source:** Jose Luis Ponton
- **Link:** https://arxiv.org/abs/2310.05215
- **Thesis fit:** Useful
- **Supports:** Animation / Implementation
- **Why it matters:**
  - 讨论 data-driven character animation 和 motion matching。
  - 虽然当前项目未必做 full motion matching，但可帮助讨论 animation graph 的扩展性问题。
  - 对 “blended poses vs scalable animation selection” 有参考价值。
- **How to use in thesis:** Animation system related work。
- **Caveat:** 不是 melee combat 专门文献；更适合做 animation system 背景。

### 5. Behavior Trees / Behavior Tree Formalization

- **Type:** paper / book area
- **Key source:** Michele Colledanchise, Petter Ögren — behavior trees research
- **Starting Link:** https://en.wikipedia.org/wiki/Behavior_tree_%28artificial_intelligence%2C_robotics_and_control%29
- **Thesis fit:** Useful
- **Supports:** Boss AI
- **Why it matters:**
  - Behavior Tree 是 game AI 和 robotics 中常见的 decision architecture。
  - 对 boss AI 的 attack pattern、phase transition、reaction logic 很适合。
  - 可以和 finite state machine 对比。
- **How to use in thesis:** AI architecture comparison：FSM vs Behavior Tree vs Utility AI。
- **Caveat:** Wikipedia 只能当入口；正式 thesis 应引用原 paper 或书。

### 6. *Artificial Intelligence for Games*

- **Type:** book
- **Authors / Source:** Ian Millington, John Funge
- **Entry point:** https://en.wikipedia.org/wiki/Artificial_intelligence_in_video_games
- **Thesis fit:** Core / Background
- **Supports:** Boss AI / State Machines / Behavior Selection
- **Why it matters:**
  - Game AI 经典教材。
  - 适合支撑 finite state machine、behavior tree、decision making、steering 等基础。
  - 可作为 boss AI chapter 的理论基础。
- **How to use in thesis:** AI chapter 的 foundational reference。
- **Caveat:** 需要找正式书籍版本信息再生成 BibTeX。

### 7. Game AI Pro / Utility AI References

- **Type:** practitioner book series
- **Source:** Steve Rabin / Game AI Pro series
- **Entry point:** https://en.wikipedia.org/wiki/Utility_system
- **Thesis fit:** Useful
- **Supports:** Boss AI / Attack Pattern Selection
- **Why it matters:**
  - Utility AI 适合 boss 根据玩家距离、血量、上次攻击、玩家行为选择攻击。
  - 可以作为 FSM 的补充：FSM 管 phase，utility score 管 attack choice。
- **How to use in thesis:** 如果 boss AI 不想只是硬编码 state machine，可以加一小节 “state machine + utility scoring”。
- **Caveat:** 更偏 industry/practitioner，不一定是强 academic source。
